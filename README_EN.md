# Cyber Threat Detection & Security Analytics

This project focuses on exploratory data analysis (EDA), network traffic analysis, and MITRE ATT&CK mapping using a SOC-oriented cybersecurity dataset.

## Project Goal

The main objectives of this project are:

- Analyze cyber attack behaviors
- Investigate network traffic characteristics
- Interpret different attack types
- Map attack behaviors to the MITRE ATT&CK framework
- Perform SOC-oriented threat analysis

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
- MITRE ATT&CK Navigator

## Analyses Performed

### Exploratory Data Analysis (EDA)

- Attack type distribution analysis
- Protocol distribution analysis
- Top destination ports analysis
- Bytes sent analysis
- Correlation analysis

### MITRE ATT&CK Mapping

Detected attack behaviors were mapped to the MITRE ATT&CK framework.

Some of the mapped techniques:

- T1110 - Brute Force
- T1046 - Network Service Scanning
- T1498 - Network Denial of Service
- T1190 - Exploit Public-Facing Application
- T1071 - Application Layer Protocol
- T1059 - Command and Scripting Interpreter
